Инструкции для проверки задания:
- Откройте три термиинала Ubuntu, активируйте виртуальное окружение.
- Войдите в директорию проекта (cd NewsPaper)
- В первом терминале запустите сервер Django (python manage.py runserver)
- Во втором создайте подключение к серверу Redis (redis-server)
- В третьем терминале запустите Celery (celery -A NewsPaper worker -l INFO)
